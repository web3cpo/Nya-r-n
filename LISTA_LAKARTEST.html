🩺 LISTA TILL LÄKAREN — Tester att begära

Boka tid hos vårdcentral. Ta med denna lista. Förklara att du vill optimera ett egenvårds-program för ledinflammation/artros och vill ha utgångsvärden + uppföljning vid 12 veckor.

🩸 BLODPROVER (begär ALLA dessa vid första besöket)

Inflammationsmarkörer (kärn)

Test Förkortning Varför
Högkänsligt CRP hs-CRP Systemisk inflammation; mål < 3 mg/L (helst < 1)
Sänka SR / ESR Komplement till CRP

Vitamin- & mineralstatus

Test Varför
25(OH)-vitamin D Mål > 75 nmol/L. Om < 50 → behov av tillskott bekräftat
Magnesium (helst RBC-magnesium om tillgängligt) Vanligt med funktionell brist
Folat + B12 Påverkar muskelmetabolism och smärt-modulering
Ferritin + Hb Järnstatus — låg-järn ger trötthet och försämrad rörelse

Differentialdiagnostik (för att utesluta annat än artros)

Test Vad det utesluter
Reumatoid faktor (RF) Reumatoid artrit (RA)
Anti-CCP RA — mer specifikt än RF
HLA-B27 Spondylartriter (om misstanke)
Urat (urinsyra) Gikt (särskilt om akuta attacker av tå/knä)
TSH + fritt T4 Hypotyreos (kan ge generell muskel/ledvärk)

Metabol baseline

Test Varför relevant
HbA1c Diabetes försämrar ledhälsa via AGE-bildning
Lipidstatus (kol, LDL, HDL, TG) Inflammation + metabol syndrom samvarierar
Kreatinin + eGFR Njurfunktion (viktigt om NSAID ska användas)
ASAT, ALAT, GT Leverstatus (curcumin metaboliseras i lever)

📷 BILDDIAGNOSTIK (begär om OA-diagnos inte redan ställd)

Undersökning När
Slätröntgen av drabbad led (knä/höft/hand) Förstahandsval — visar Kellgren-Lawrence-grad I–IV
MR (magnetkamera) Vid avvikande symtom, ung patient, eller misstanke om annat (meniskskada, osteonekros)
Ultraljud av leden För synovit-bedömning, billigt & strålningsfritt

🧴 RECEPTBELAGT FRÅN LÄKARE (Tier 0 — diskutera vid första besöket)

Dessa är OARSI-rekommenderade och kräver oftast recept eller läkares råd.

Preparat Indikation Form
Diklofenak gel 1–2 % (Voltaren, Diklofenak) Topikalt mot smärta — strongly recommended Receptfritt på apotek, men diskutera dosering
Topikal capsaicin (kräm/plåster) Vid hand-OA eller fokal smärta Receptbelagt i högre koncentration
NSAID per os (ibuprofen, naproxen, etoricoxib) Endast vid behov, kortast möjliga tid Diskutera GI/CV-risk
Acetaminofen/paracetamol OARSI 2019 säger "conditional NOT recommended" — låg effekt Endast vid kontraindikation mot NSAID
Intraartikulär kortison Vid akut svår synovit, max 3–4 ggr/år Görs av ortoped/reumatolog
Intraartikulär hyaluronsyra Vid otillräcklig respons Görs av specialist

💊 DISKUTERA OM PÅGÅENDE MEDICINERING

Lista ALLT du tar (även receptfritt) och fråga om interaktioner.

Om du tar… Diskutera…
Warfarin / Eliquis / Xarelto / Pradaxa Curcumin, omega-3 högdos, älggräs, devil's claw — alla kan öka blödningsrisk
Metformin Curcumin kan svagt potentiera blodsockereffekt
Statiner Curcumin är CYP3A4-substrat — kan påverka
SSRI / antidepressiva SAMe + SSRI = teoretisk serotonergt syndrom-risk
Cytostatika / immunsuppressiva All supplement-användning ska godkännas
Kortison (oralt) Diskutera nedtrappning vid bra respons på protokoll

📋 FRÅGOR ATT STÄLLA LÄKAREN

  1. "Kan jag få min Kellgren-Lawrence-grad dokumenterad i journalen?"
  2. "Är det artros eller kan det vara reumatoid artrit / gikt / spondylartros?"
  3. "Kan jag få remiss till fysioterapeut specialiserad på artros?"
  4. "Om jag testar Tier A-supplement i 12 v utan effekt — är artroskopi/protes-utredning aktuell?"
  5. "Vid kraftig flare — kan jag få kortison-injektion intraartikulärt?"
  6. "Kan jag få prov-ordination på topikal capsaicin/diklofenak?"
  7. "Önskar uppföljning av CRP, ESR, 25(OH)D vid 12 veckor för att utvärdera effekt"

🩻 VID UTREDNING — REMISS TILL SPECIALIST KAN BEHÖVAS

Specialist När?
Reumatolog Vid förhöjt RF/CCP eller systemiska symtom
Ortoped Vid KL-grad III–IV eller funktionsnedsättning som påverkar dagligt liv
Fysioterapeut ALLA — Level 1-evidens för rörelse-program
Dietist Om viktnedgång behövs eller GI-känslighet för supplement
Smärtklinik Vid kronisk smärta som inte svarar på protokoll
